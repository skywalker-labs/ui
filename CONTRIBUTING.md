# Contributing

Welcome to Skywalker Labs UI! We sincerely appreciate your interest in contributing. By doing so, you can help us sustain a high-quality frontend implementation package for Laravel.

## How to Contribute

Whether it's bug fixes, performance tests, or new CSS presets, your input is valuable.

1. **Fork the Repository:** Create a personal fork and start a new branch for your feature or fix.
2. **Adhere to Standards:** Ensure you write PSR-12 compliant code. If writing CSS/JS scaffolding, follow modern ES6/Sass syntax closely.
3. **Describe Your Changes:** Be extremely specific in your Pull Request about what problem your code fixes or what new feature is provided.
4. **Write Tests:** All Pull Requests should include corresponding tests demonstrating that your fix or feature behaves correctly.

## Running Tests

To run the local test suite using Pest or PHPUnit:

```bash
composer test
```

*Or manually format it using PHPUnit:*

```bash
vendor/bin/phpunit
```

## Security Vulnerabilities

If you happen to find a vulnerability within the project, please respect the [Security Policy](SECURITY.md) instead of opening a public issue. 

> *Your involvement helps shape the tools that empower developers to build incredible things. We look forward to evaluating your Pull Requests!*
