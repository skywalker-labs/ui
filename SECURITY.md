# Security Policy

If you discover any security-related issues, or zero-day vulnerabilities in the `skywalker-labs/ui` framework infrastructure, please report them immediately.

## Supported Versions

Currently, we exclusively provide active security monitoring and immediate patching for our primary stable release lines.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.0  | :white_check_mark: |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

Please **DO NOT** use the public issue tracker to disclose security vulnerabilities. 

Instead, send a direct email with a Proof of Concept (PoC) to our lead developer:  
**Email:** `ermradulsharma0@gmail.com`

We will actively work with you to verify the vulnerability, prepare a patch, and coordinate a secure disclosure timeframe. Thank you for your responsible engagement.
